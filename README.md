# Java Learning Project

This repository contains a collection of Java programs and examples, organized by core topics.  
The structure is arranged to help learners start from basic concepts and gradually move to advanced topics. Each folder contains well-commented and structured code for better understanding.

---

## **Sequence of learning & Topics (Basic â†’ Advanced)**

1. **[tokens](tokens/)**  
   Learn about Java tokens: keywords, identifiers, literals, operators, separators, and comments with clean, beginner-friendly examples.

2. **[data_types](data_types/)**  
   Examples and explanations of Java data types, including primitive types, wrapper classes, literals, and type casting.

3. **[switch-statement](switch-statement/)**  
   Examples of `switch` statements with comments explaining how they work and when to use them.

4. **[operators](operators/)**  
   Demonstrates all types of operators in Java: arithmetic, relational, logical, assignment, bitwise, ternary, and more.

5. **[loops](loops/)**  
   Examples of loops in Java: `for`, `while`, `do-while`, and nested loops with comments and explanations on usage.

6. **[arrays](arrays/)**  
Contains examples of Java arrays, including 1D arrays, 2D arrays, and array operations with proper explanations.
7. **[collection_frameworks](collection_frameworks/)**  
   Examples of Java Collection Framework including ArrayList, Vector, LinkedList, Stack, Queue, Set (HashSet, LinkedHashSet, TreeSet) with constructors, methods, and comments.

8. **[out](out/)**  
   Compiled output folder generated by IDE. No source code is stored here.

---

## **How to Run**

1. Open the project in your preferred IDE (IntelliJ IDEA, Eclipse, etc.).
2. Navigate to the folder of the topic you want to run.
3. Compile and run the Java file using the IDE's run option, or via terminal:

```bash
javac ClassName.java
java ClassName

```
